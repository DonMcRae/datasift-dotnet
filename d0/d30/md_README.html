README

This example demonstrates how to use the Historics and Push delivery parts of the library. Please note that this is just an example and as such the UI is not very well polished, but it does show how to use the main parts of the API.

This is the official .NET library for accessing Datasift. See the examples project for some simple example usage.

Note that we use git flow to manage development.

Requirements

  • Compiler for .NET 3.5+

Known issues

  • Only two connections can be created to the streaming server. Further attempts to establish a connection will hang.
